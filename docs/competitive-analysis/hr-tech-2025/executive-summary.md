3. КОНКУРЕНТЫ УЖЕ ДВИЖУТСЯ

   └─ → Ваше окно конкурентного преимущества 12-18 месяцев

## Риски и как их управлять

### Главные риски реализации

```
РИСК 1: Низкое adoption (сотрудники не используют)
─────────────────────────────────────────────────────
Probability: Среднее (40%)
Impact: ВЫСОКОЕ (может убить весь ROI)

Mitigation:
✓ Сильный change management (выделить ресурсы)
✓ Manager champion программа (Top-down push)
✓ Quick wins early (покажите первые успехи)
✓ Internal promotion (обычно требуется)
✓ Выбрать платформу с хорошим UX (LATTICE, FUEL50)

РИСК 2: Плохая качество данных (skills не accurate)
──────────────────────────────────────────────────────
Probability: Высокое (70%)
Impact: Среднее (рекомендации будут неправильные)

Mitigation:
✓ Data cleanup ПЕРЕД реализацией (важно!)
✓ Выбрать платформу с AI inference (FUEL50, Eightfold)
✓ Phased rollout (начать с clean data teams)
✓ Continuous refinement (это ожидается)

РИСК 3: Более долгая реализация чем ожидается
───────────────────────────────────────────────
Probability: Среднее (35%)
Impact: Среднее (delays ROI на несколько месяцев)

Mitigation:
✓ Выбрать быструю платформу (FUEL50: 42% live в 1-3 мес)
✓ Minimal customization (используй defaults)
✓ Phased rollout (быстрый PoC, потом full rollout)
✓ Dedicated project team (выделить ресурсы)

РИСК 4: Технический, интеграционный
──────────────────────────────────────
Probability: Среднее (30%)
Impact: Среднее (delay + extra costs)

Mitigation:
✓ Pre-assess HRIS integration capability
✓ Вовлечь IT на раннем этапе
✓ Test integration в PoC фазе
✓ Pre-built connectors лучше (FUEL50, Gloat, Eightfold)
```
