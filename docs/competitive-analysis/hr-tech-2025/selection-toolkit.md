## Часть 3: Матрица решений по сценариям

### 3.1 Быстрый выбор платформы

**Используйте этот тест для быстрого определения типа платформы:**

```
START
  ↓
1. Ваша компания меньше 2,000 сотрудников?
   ├─ ДА → перейти к 2A
   └─ НЕТ → перейти к 2B

2A. КОМПАНИИ МЕНЕЕ 2,000 СОТРУДНИКОВ
   ↓
   Главный фокус?
   ├─ Performance Management & Engagement
   │  └─ → РЕКОМЕНДАЦИЯ: LATTICE ($11/user/месяц)
   │     Преимущества: прозрачная цена, отличный AI
   │
   ├─ Career Development & Pathing (vs управление производительностью)
   │  └─ → РЕКОМЕНДАЦИЯ: TALENTGUARD ($3/user/месяц*)
   │     Преимущества: доступно, хорошее pathing
   │
   └─ Internal Mobility (есть ли бюджет для enterprise решения?)
      ├─ ДА (есть бюджет)
      │  └─ → РЕКОМЕНДАЦИЯ: FUEL50
      │     Преимущества: лучший skills management
      │
      └─ НЕТ (ограниченный бюджет)
         └─ → РЕКОМЕНДАЦИЯ: SKYHIVE via Cornerstone ($5-10/user)
            Преимущества: доступно, хорошо для skills

2B. КОМПАНИИ БОЛЕЕ 2,000 СОТРУДНИКОВ
   ↓
   Какой главный need?
   ├─ Internal Mobility Only
   │  └─ → Перейти к 3A
   │
   ├─ Hiring + Internal Mobility Both
   │  └─ → РЕКОМЕНДАЦИЯ: EIGHTFOLD AI
   │     Преимущества: comprehensive, strong AI
   │
   └─ Performance + Mobility (Hybrid)
      └─ → РЕКОМЕНДАЦИЯ: Комбинация Lattice + Fuel50
         Преимущества: лучшее в обоих

3A. INTERNAL MOBILITY PLATFORM (ENTERPRISE)
   ↓
   Приоритет номер 1?
   ├─ Лучшие Skills Management & Accuracy
   │  └─ → РЕКОМЕНДАЦИЯ: FUEL50 ⭐
   │     Причины: 9.2/10 skill mapping, 10/10 product direction
   │     ROI: Самый быстрый (42% live в 1-3 месяца)
   │     Результаты: 60% churn reduction, 65% lateral movement
   │
   ├─ Скорость Реализации & PoC
   │  └─ → РЕКОМЕНДАЦИЯ: FUEL50 ⭐⭐
   │     Причины: Fastest implementation
   │
   ├─ Размер клиентов (крупные enterprise) + Reputation
   │  └─ → РЕКОМЕНДАЦИЯ: GLOAT
   │     Причины: Unilever, HSBC, Schneider Electric клиенты
   │     Преимущества: Проверенная Workforce Graph
```
