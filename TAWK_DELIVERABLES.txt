QUICK SETUP INSTRUCTIONS
================================================================================

Step 1: CREATE ACCOUNT (5 minutes)
  - Visit https://www.tawk.to
  - Sign up (no credit card required)
  - Create first website property
  - Go to: Settings > Channels > Website
  - Extract Property ID and Widget ID from embed code

Step 2: CONFIGURE ENVIRONMENT (2 minutes)
  - Open or create .env.local
  - Add: NEXT_PUBLIC_TAWK_PROPERTY_ID=your_property_id
  - Add: NEXT_PUBLIC_TAWK_WIDGET_ID=your_widget_id
  - Save and close

Step 3: TEST LOCALLY (5 minutes)
  - Run: pnpm dev
  - Open: http://localhost:3001
  - Widget should appear in bottom-right corner
  - Check browser console for [Tawk.to] initialization logs
  - Click widget to test chat functionality

Step 4: DEPLOY (depends on environment)
  - Staging: Set env vars in Vercel preview environment
  - Production: Set env vars in Vercel production environment
  - Docker: Use docker-compose with environment variables
